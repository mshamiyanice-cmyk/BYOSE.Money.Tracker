## Overview
This application serves as the central financial hub for BYOSE Tech, allowing for:
- **Shared Company Ledger:** All financial data (Inflows, Outflows, Overdrafts) is centralized.
- **Role-Based Access:** 
  - **Admins:** Full Read/Write access.
  - **Viewers:** Read-Only access.
- **Smart Features:** overdrdaft management, partial settlements, and intuitive dashboards.

## Getting Started

### Prerequisites
- Node.js
- Firebase Project configured

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mshamiyanice-cmyk/BYOSE.Money.Tracker.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Tech Stack
- React + TypeScript + Vite
- Tailwind CSS
- Firebase (Auth, Firestore)
